# Contact Book

## Description

Contact Book is a simple Python application that allows users to store and manage contact information. Users can add, view, search, and delete contacts through a menu-driven interface.

## Features

* Add new contacts
* View all contacts
* Search for a contact
* Delete contacts
* Exit the application

## Technologies Used

* Python 3

## How to Run

1. Make sure Python 3 is installed on your computer.
2. Download or clone the project files.
3. Open a terminal in the project folder.
4. Run the program using:

python contact_book.py

## Learning Outcomes

This project helped practice:

* Variables
* Lists and Dictionaries
* Loops
* Conditional Statements
* Functions
* User Input Handling
* Basic Debugging
